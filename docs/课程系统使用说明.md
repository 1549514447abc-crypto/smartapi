## 📊 数据库表结构

### 1. course_settings（课程设置表）
```sql
- id: 主键
- title: 课程标题（React全栈开发实战课程）
- subtitle: 副标题
- description: 课程描述
- original_price: 原价（¥599）
- current_price: 现价（¥299）
- instructor_name: 讲师姓名（Alex 老师）
- instructor_bio: 讲师简介
- highlights: 课程亮点（JSON数组）
- outline: 课程大纲（JSON数组）
- wechat_qr_image: 微信二维码路径
```

### 2. course_lessons（试听课程表）
```sql
- id: 主键
- title: 课程标题
- video_path: 视频文件路径
- duration: 时长（秒）
- sort_order: 排序序号
```

**当前状态：** 已插入10节试听课，所有课程暂时都使用 `01.mp4` 作为占位视频。

### 3. course_orders（订单表）
```sql
- id: 主键
- order_no: 订单号
- user_id: 用户ID
- course_title: 课程标题
- amount: 支付金额
- status: 订单状态（pending/completed/cancelled）
- admin_note: 管理员备注
```

## 📝 后续需要做的事情

### 1. 上传微信二维码 ⚠️
**当前状态：** 微信二维码路径为NULL，支付页面会显示"二维码暂未配置"

**解决方法：**
```sql
-- 更新微信二维码路径（假设您的二维码文件名为 wechat-qr.jpg）
UPDATE course_settings
SET wechat_qr_image = '/uploads/course/images/wechat-qr.jpg'
WHERE id = 1;
```

**步骤：**
1. 将您的微信二维码图片保存到：
   `D:\code-program\smartapi\backend\uploads\course\images\wechat-qr.jpg`
2. 执行上面的SQL更新数据库
3. 刷新支付页面，二维码就会显示了

---

### 2. 上传真实的试听视频 ⚠️
**当前状态：** 10节试听课都使用 `01.mp4` 作为占位

**解决方法：**
```bash
# 将您的视频文件复制到lessons目录
cp /path/to/your/lesson-02.mp4 D:/code-program/smartapi/backend/uploads/course/lessons/02.mp4
cp /path/to/your/lesson-03.mp4 D:/code-program/smartapi/backend/uploads/course/lessons/03.mp4
# ... 依次类推

# 然后更新数据库
UPDATE course_lessons SET video_path = '/uploads/course/lessons/02.mp4' WHERE id = 2;
UPDATE course_lessons SET video_path = '/uploads/course/lessons/03.mp4' WHERE id = 3;
# ... 依次类推
```

## 🔑 API接口文档

### 用户端接口

#### 1. 获取课程信息
```http
GET /api/course/info
Response: { 课程完整信息 }
```

#### 2. 获取试听课列表
```http
GET /api/course/lessons
Response: { lessons: [...] }
```

#### 3. 获取视频流
```http
GET /api/course/lessons/:id/video
Response: 视频流（支持Range请求）
```

#### 4. 创建订单（需要登录）
```http
POST /api/course/orders
Headers: { Authorization: Bearer <token> }
Response: { order_no, course_title, amount, wechat_qr_image }
```

#### 5. 获取我的订单（需要登录）
```http
GET /api/course/orders/my
Headers: { Authorization: Bearer <token> }
Response: { orders: [...] }
```

#### 6. 获取订单详情（需要登录）
```http
GET /api/course/orders/:orderNo
Headers: { Authorization: Bearer <token> }
Response: { 订单详情 + 微信二维码 }
```

## 💡 常见问题

### Q1: 视频无法播放？
**A:** 检查视频文件是否存在：
```bash
ls D:/code-program/smartapi/backend/uploads/course/lessons/01.mp4
```

### Q2: 支付页面不显示二维码？
**A:** 需要上传微信二维码图片并更新数据库：
```sql
UPDATE course_settings
SET wechat_qr_image = '/uploads/course/images/wechat-qr.jpg'
WHERE id = 1;
```

### Q3: 如何修改课程价格？
**A:** 直接修改数据库：
```sql
UPDATE course_settings SET current_price = 199.00 WHERE id = 1;
```

### Q4: 如何查看所有订单？
**A:** 执行SQL查询：
```sql
SELECT o.order_no, u.username, o.amount, o.status, o.created_at
FROM course_orders o
JOIN users u ON o.user_id = u.id
ORDER BY o.created_at DESC;
```
