### 课程描述
```
本课程专为零基础学员和想要系统学习React的开发者设计。课程从最基础的JavaScript ES6语法开始，逐步深入到React核心概念、Hooks进阶用法、状态管理、路由配置，再到Node.js后端开发，最终带你完成一个完整的全栈项目——在线课程销售平台。

课程采用"理论+实战"的教学模式，每个知识点都配有详细的代码演示和练习题。你将学会如何使用React + TypeScript + Ant Design构建现代化的前端应用，使用Express + MySQL搭建RESTful API后端服务。

课程包含65节正式课程，总时长超过40小时，覆盖从入门到精通的完整学习路径。学完后你将具备独立开发中小型全栈项目的能力，可以胜任初中级前端工程师岗位。
```

## 三、课程大纲

```
第1章：JavaScript ES6核心语法（共8节课）
  - let/const、解构赋值、箭头函数
  - Promise、async/await异步编程
  - 模块化开发

第2章：React基础入门（共10节课）
  - JSX语法与组件化思想
  - 函数组件与类组件
  - Props与State状态管理
  - 事件处理与表单控制

第3章：React Hooks进阶（共12节课）
  - useState、useEffect详解
  - useContext、useReducer状态管理
  - useMemo、useCallback性能优化
  - 自定义Hooks封装

第4章：React Router路由管理（共6节课）
  - 路由配置与嵌套路由
  - 路由传参与导航守卫
  - 懒加载与代码分割

第5章：TypeScript实战应用（共8节课）
  - TypeScript基础类型
  - 接口与泛型
  - React + TypeScript最佳实践

第6章：Ant Design组件库（共5节课）
  - 表单组件与数据展示
  - 布局与导航
  - 自定义主题配置

第7章：Node.js后端开发（共10节课）
  - Express框架搭建
  - RESTful API设计
  - MySQL数据库操作
  - JWT身份认证

第8章：全栈项目实战（共6节课）
  - 项目架构设计
  - 前后端联调
  - 文件上传与视频播放
  - 部署上线

总计：65节课，40+小时
```
