## 🚀 快速开始指南

### 第一步：安装MySQL和创建数据库

```bash
# 1. 确保MySQL已安装并启动
# Windows: 在服务中启动MySQL服务
# Mac: brew services start mysql

# 2. 登录MySQL
mysql -u root -p

# 3. 创建数据库和用户
```

```sql
-- 创建数据库
CREATE DATABASE smartapi_dev DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

-- 创建用户（可选，建议创建专用用户）
CREATE USER 'smartapi_user'@'localhost' IDENTIFIED BY 'SmartAPI@2024!';
GRANT ALL PRIVILEGES ON smartapi_dev.* TO 'smartapi_user'@'localhost';
FLUSH PRIVILEGES;

-- 退出
EXIT;
```

---

### 第二步：初始化Node.js项目

```bash
# 创建后端项目目录
mkdir smartapi-backend
cd smartapi-backend

# 初始化项目
pnpm init

# 安装依赖
pnpm add express cors dotenv mysql2 sequelize
pnpm add jsonwebtoken bcryptjs axios
pnpm add -D typescript @types/node @types/express @types/cors
pnpm add -D @types/jsonwebtoken @types/bcryptjs
pnpm add -D sequelize-cli ts-node nodemon

# 初始化TypeScript
npx tsc --init
```

**1. 创建 `.sequelizerc` 文件（项目根目录）**


## 🐛 常见问题

### 问题1: MySQL连接失败
```
Error: connect ECONNREFUSED 127.0.0.1:3306
```
**解决方案**:
- 确认MySQL服务已启动
- 检查端口是否正确（默认3306）
- 检查用户名和密码

### 问题2: 字符集错误
```
Error: ER_CHARSET_NOT_FOUND
```
**解决方案**:
```sql
-- 修改数据库字符集
ALTER DATABASE smartapi_dev CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 问题3: Sequelize CLI命令找不到
```
sequelize-cli: command not found
```
**解决方案**:
```bash
# 使用npx执行
npx sequelize-cli db:migrate

# 或者全局安装
npm install -g sequelize-cli
```
