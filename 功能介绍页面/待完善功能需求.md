## 六、技术备注

### Python 部署方案
使用 **paramiko** 库实现 SSH 连接和文件上传：

```python
# 核心依赖
pip install paramiko

# 功能
- SSH 远程命令执行
- SFTP 文件上传
- 支持 Windows 环境
```

### 需要新增的数据库字段（如果没有）
- `users.points` - 用户积分余额
- `points_logs` - 积分变动记录表
