# 🚀 微信登录功能 - 快速开始指南

欢迎！这份指南将帮你在30分钟内完成微信登录功能的部署。

---

## 📝 准备清单

在开始之前，请确认你已经准备好：

- ✅ 腾讯云服务器（已有）
- ✅ 域名和网站（已有）
- ✅ 微信公众号（已有）
- ✅ SSH 访问服务器的权限
- ✅ 微信公众号管理员权限

---

## 🎯 第一步：获取微信配置（5分钟）

### 1. 登录微信公众平台
访问：https://mp.weixin.qq.com/

### 2. 获取 AppID 和 AppSecret
```
导航路径：设置与开发 → 基本配置 → 开发者ID
```

记录以下信息：
- **AppID**: 类似 `wx1234567890abcdef`
- **AppSecret**: 点击"重置"生成，⚠️ 只显示一次，务必保存！

### 3. 检查公众号类型
- ✅ **已认证服务号**：可以直接使用
- ❌ **订阅号或未认证**：需要先升级/认证

---

## 🔧 第二步：配置本地环境（5分钟）

### 1. 安装依赖
在项目目录下运行：

```bash
npm install
```

这会安装新增的 `dotenv` 依赖包。

### 2. 创建配置文件
复制 `env.example` 文件并重命名为 `.env`：

**Windows:**
```bash
copy env.example .env
```

**Mac/Linux:**
```bash
cp env.example .env
```

### 3. 填写配置
用文本编辑器打开 `.env` 文件，填入你的信息：

```env
# 微信公众号配置
WECHAT_APP_ID=wx你的实际AppID
WECHAT_APP_SECRET=你的实际AppSecret
WECHAT_REDIRECT_URI=https://你的域名.com/wechat/callback

# 服务器配置
PORT=3001
NODE_ENV=production
```

⚠️ **重要**：不要把 `.env` 文件提交到 Git！

### 4. 修改前端配置
打开 `js/config.js` 文件，第 10 行：

```javascript
apiBaseUrl: 'https://你的域名.com',  // 改成你的实际域名
```

---

## 🌐 第三步：配置微信公众号（5分钟）

### 1. 设置授权回调域名
```
导航路径：设置与开发 → 接口权限 → 网页服务 → 网页授权 → 网页授权域名
```

填写你的域名（不带 http:// 和端口号）：
```
your-domain.com
```

### 2. 下载验证文件
- 微信会提供验证文件：`MP_verify_xxxxxxxxxx.txt`
- 下载后上传到你的**网站根目录**
- 确保可以通过 `http://your-domain.com/MP_verify_xxxxxxxxxx.txt` 访问

### 3. 完成验证
在微信后台点击"确认"完成验证。

---

## ☁️ 第四步：部署到服务器（10分钟）

### 方法A：使用自动部署脚本（推荐）

#### 1. 修改部署脚本配置
打开 `deploy.sh` 文件，修改前3行：

```bash
SERVER_USER="root"              # 你的服务器用户名
SERVER_HOST="123.456.789.123"   # 你的服务器IP
SERVER_PATH="/www/jianying-server"
```

#### 2. 执行部署
**Mac/Linux:**
```bash
chmod +x deploy.sh
./deploy.sh
```

**Windows:** 
推荐使用 Git Bash 或者使用手动部署方法。

### 方法B：手动部署

#### 1. 连接到服务器
```bash
ssh root@你的服务器IP
```

#### 2. 创建部署目录
```bash
mkdir -p /www/jianying-server
cd /www/jianying-server
```

#### 3. 上传文件
从本地上传以下文件到服务器：
- `server.js`
- `package.json`
- `.env`
- `ecosystem.config.js`

可以使用 FTP 工具（如 FileZilla）或命令行：

```bash
# 在本地执行（替换为你的服务器IP）
scp server.js package.json .env ecosystem.config.js root@服务器IP:/www/jianying-server/
```

#### 4. 安装依赖
在服务器上执行：
```bash
cd /www/jianying-server
npm install --production
```

#### 5. 安装 PM2
```bash
npm install -g pm2
```

#### 6. 启动服务
```bash
mkdir -p logs
pm2 start ecosystem.config.js
pm2 save
pm2 startup
```

#### 7. 查看状态
```bash
pm2 status
pm2 logs jianying-api
```

## 🔒 第五步：配置 Nginx 和 HTTPS（5分钟）

### 1. 申请 SSL 证书
在腾讯云控制台申请免费证书：
```
腾讯云控制台 → SSL证书 → 申请免费证书
```

### 2. 下载证书
- 选择 **Nginx** 格式
- 下载后得到两个文件：`.crt` 和 `.key`

### 3. 上传证书到服务器
```bash
# 在服务器上执行
mkdir -p /etc/nginx/ssl

# 从本地上传证书
scp your-domain.com.crt your-domain.com.key root@服务器IP:/etc/nginx/ssl/
```

### 4. 配置 Nginx
```bash
# 在服务器上执行
nano /etc/nginx/sites-available/jianying-api
```

复制 `nginx.conf.example` 文件的内容，并修改：
- 所有 `your-domain.com` 改为你的实际域名
- 证书路径改为实际路径
- 网站根目录改为实际目录

### 5. 启用配置
```bash
ln -s /etc/nginx/sites-available/jianying-api /etc/nginx/sites-enabled/
nginx -t
systemctl reload nginx
```

## ✅ 第六步：测试登录功能（2分钟）

### 1. 启动客户端
在本地项目目录：
```bash
npm start
```

### 2. 测试登录
1. 打开应用后进入 **"个人中心"**
2. 点击 **"微信扫码登录"**
3. 使用微信扫描二维码
4. 在手机上确认登录
5. 查看是否成功显示用户信息

### 3. 调试方法
如果遇到问题，按 `F12` 打开开发者工具查看：
- Console 标签：查看 JavaScript 错误
- Network 标签：查看 API 请求状态

服务器端查看日志：
```bash
pm2 logs jianying-api
tail -f /var/log/nginx/error.log
```

## 🆘 遇到问题？

### 常见问题排查

**问题1：二维码无法生成**
```bash
# 检查服务器是否运行
pm2 status

# 查看日志
pm2 logs jianying-api

# 测试 API
curl https://你的域名.com/api/wechat/qrcode
```

**问题2：扫码后没反应**
- 检查微信后台的回调域名配置
- 确认 HTTPS 证书正常
- 查看 Nginx 错误日志

**问题3：提示 redirect_uri 错误**
- 确认 `.env` 中的回调地址与微信后台一致
- 重启服务：`pm2 restart jianying-api`
